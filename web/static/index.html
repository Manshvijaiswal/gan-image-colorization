The pipeline

Three stages, the same order every darkroom print follows.

Chroma's training process mirrors traditional photo development — expose the base image, develop detail into it, then fix the result so it holds. Each stage below is a real step in the network's forward pass.

01 — EXPOSE

Split the image into light and color

Every training photo is converted to LAB color space. The L channel (lightness) becomes the network's only input — identical to a grayscale photograph. The a/b channels, which hold all the color, are set aside as the target to predict.

RGB → LAB
02 — DEVELOP

The generator predicts color

An 8-layer U-Net receives the L channel and predicts a and b. Skip connections carry fine edge detail straight from encoder to decoder, so color lands precisely at object boundaries rather than bleeding across them.

U-Net Generator
03 — FIX

The discriminator holds it to the light

A PatchGAN discriminator inspects overlapping 70×70 patches and rejects anything that looks implausible, pushing the generator toward vivid, realistic color rather than the safe, desaturated average.
